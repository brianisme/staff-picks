# staff-picks
Vimeo Staff Picks Showcase

# Prereqs

```bash
# installing node
$ sudo apt-get install -y nodejs
# installing bower
$ npm install -g bower
# installing gulp
$ sudo npm install gulp -g
```

# Run in local
```bash
$ gulp serve
```

# Run tests
```bash
$ gulp test
```
